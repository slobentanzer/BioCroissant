# biocroissant package

**Purpose:** Core package providing parser, validation, converters, and CLI for BioCroissant.

**Current state:** Skeleton only; no implementation yet. This layout is for collaborator feedback.

**Planned content:** `core` (parser, schema loading), `validation` (schema and semantic checks), `converters` (OMOP first, then FHIR, imaging, omics as needed), and `cli` (validate, convert, describe). Implementation will be added in a later phase, possibly ported from the biocroissant-to-omop prototype with improvements.
